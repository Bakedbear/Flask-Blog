
# Flask Blog

## Features

- [x] User registration
- [x] Login
- [x] Logout
- [x] Create Posts
- [x] Edit post
- [x] Read post 
- [x] Delete post
- [x] See all posts from a particular user
- [x] Account Page
- [x] Update Profile picture
- [ ] Password Reset 
- [ ] Post reactions 


## Requirements

- Flask - 2.3.2
- Flask_SQLAlchemy - 2.5.1
- SQLAlchemy - 2.0.17 or newer
- Werkzeug 2.3.6
- Passlib
- WTForms
- Python 3.11.4 or newer
